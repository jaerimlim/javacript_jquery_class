자바스크립트 내장객체_5윈도우(window)_브라우저 객체

브라우저 객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체이다.

Window 객체

window객체는 브라우저 객체 중 '최상위 객체'로서 현재 실행되고 있는 브라우저창을 나타낸다.

Window객체의 속성과 메서드

Window객체의 속성(property)종류
document
frameElement
innerHeight
innerWidth
location
name
screenX
screenY
scrollX, scrollY
sessionStorge
Window객체의 메서드(method)종류
alert(); - 경고창 띄우기
blur(); - 현재창에서 포커스를 제거
close(); - 현재창을 닫기
confirm(); - 확인/취소 버튼이 있는 확인창을 표시.
focuse(); - 현재창에 포커스(초점)를 부여함
moveBy(); - 현재창을 지정한 크기만큼 이동
moveTo(); - 현재창을 지정한 좌표로 이동
open(); - 지정한 문서를 열기
prompt(); - 사용자가 입력 가능한 프롬프트창을 활성화함.
resizeTo(); - 동적으로 브라우저 창의 크기를 조절
scroll(); - 문서의 특정위치로 스크롤 합니다.
sizeToContent(); - 내용에 맞게 창의 크기를 조절
stop(); - 로딩중지

모달윈도(modal window)
모달윈도(modal window)는 사용자 인터페이스(UI)디자인 개념에서 자식 윈도에서 부모 윈도우로 돌아가기 전에 사용자의 상호동작을 요구하는 창을 말한다.
모달윈도는 부모창을 배경으로 어둡게하고, 자식요소가 화면의 중앙에 배치하도록 하며, 오른쪽 상단에 '닫기'버튼을 삽입하여 닫을 수 있도록 한다.

    getElementById()는 무엇인가?
    id를 통해 엘리먼트를 반환한다. 만약 document에 구체적인 ID의 엘리먼트가 없다면 null을 반환한다.

    querySelector()는 무엇인가?s
    select의 구체적이 그룹과 일치하는 document안 첫번째 엘리먼트를 반환한다. 일치하는게 없으면 null반환한다.

    querySelectorAll() : document안에 있는 엘리먼트 모두 반환
